# desafio
Desafio Angular 6

Requerimientos:
-Node version >= 8.9
-Angular 6
-Angular Material 6

Instrucciones:

1-Clonar repositorio
2-Dentro de la carpeta del repositorio, instalar angular 6: npm install -g @angular/cli
3-Correr npm install 
4-Correr aplicacion angular con el comando ng serve --open
